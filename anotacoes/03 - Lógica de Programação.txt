# Lógica de Programação

Definição
	Forma de pensamento estruturada para resolver um problema
	Organização e planejamento das instruções, assertivas em 
		um algoritmo, a fim de viabilizar a implantação de 
		um programa
	
Técnicas de lógica
	Linear - não tem vinculo
		execução sequenciada
	Estruturada - segue uma condição
	Modular - partes independentes
		simplificação
		decompor problema
		verificação modulo
